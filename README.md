# CSCI-5408-W22-project



## A Menu Driven Distributed Database deployed on GCP VM Instance

The menu Driven program includes modules
1. User Registration.
2. User Login.
3. Write Queries
4. Export
5. Data Model
6. Analytics
7. Logout


## Run the Database Program
```
mvn clean install
```
